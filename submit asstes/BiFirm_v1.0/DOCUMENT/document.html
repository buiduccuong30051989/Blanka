Folder Structure

When you unpack downloaded archive from ThemeForest.net you'll get folder containing 2 folders.

Here they are, sorted alphabetically:

  • Documentation - Documentation for BIFIRM HTML site template

  • HTML - Main folder for template

    Once you open this folder you will see:

    • css folder - include all css that's i used in this template
    • js folder - include all js
    • fonts folder - include all font - in this template i used line-awesome icons
    • images folder - include all images - in this product, almost images is placeholder image
    • public folder - include some assets need for template, in this case, this is PhotoSwipe assets (images, fonts...)

CSS Files

All css files located at css folder.

  • css/main.css - Theme main CSS files

  • css/extention/*.css include all css plugins

  • css/color/*.css to change color themes, just replace color in this file

    You can go to demo link to choose and test your color you need, what does it look like, what color code you need...

    To apply colors, simple add css or colors below main.css in head tag

    <link href="css/main.css" rel="stylesheet">
    <link href="css/colors/colors.css" rel="stylesheet">

JavaScript

All JS files located at js folder.

  • js/extention/*.js include all plugins

  • js/*.js are functions for each components

  • Plugins i use in this template:
    • Bootstrap - Frontend FrameWork
    • Animsition - For smooth preload
    • ChartJs - For chart components
    • Isotope - for filter & mansory grid - use for cases components
    • Owl carousel - use for slider block or images
    • Photoswipe - use for slider modal images
    • Mmenu Light - use for mobile menu
